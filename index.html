GUESS THE GAME NIGHT WORD

Hint: Like prepared food or Nate on chess night
Alright, this week is a little different - we're doing chess puzzles!
The answer this week is a single word. You will solve the puzzles to learn the letters of the word.

Each puzzle has 1 correct move, which can be represented via algebraic chess notation.
Over the course of the 4 puzzles, the alphabetical letters in the notation of the correct moves will spell out the answer!

***I made a mistake here - notation for a capture includes an 'x' - this is not part of the answer.
Only the letters corresponding to the pieces (K, Q, B, N, R), and the board squares (A-H) are part of the answer. Sorry!***

Below is an example.
In this position we can play checkmate with the move 're8#', moving the rook to the back rank. The King has no squares to move to, and no pieces can block, so it's mate in 1.
From this puzzle we would get the letters 'R' and 'E', meaning the first 2 letters of the answer would be RE.
The notation is pretty intuitive, but if you're confused take a look at the wikipedia article linked above.

Some tips:
  • Assume white to move for all puzzles
  • The puzzles spell the word out in order
  • There is a single, obvious best move in each position
  • The word is 6 letters long, and there are 4 puzzles
    • Good luck!
  • Pawn moves: list the square they move to (i.e. c3, b5)
  • Piece moves: letter (k - king, q - queen, b - bishop, n - knight, r - rook) + square (i.e. bc3, qh2)
  • Pawn captures: put an 'x' between the file (column) and the target square (i.e. exd4, fxg7)
  • Piece captures: put an 'x' between the piece and the target square (i.e. bxc3, qxh2)
  • Checkmate: Add an '#' after the move (i.e. re7#, bxh3#)
  • Castling: Kingside castling - O-O, queenside castline - O-O-O
